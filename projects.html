9.7M+ Sq Km Coverage Data
56 Geographic Regions
30+ DAX Measures
15-40% Deployment Gaps Identified

Key Achievements

  • Built dual-platform analytics solution using Power BI and Python (Plotly/Dash) to analyze 9.7M+ sq km coverage data across 56 geographic regions, enabling stakeholders to identify coverage gaps and deployment opportunities
  • Designed and implemented 30+ DAX measures including statistical analyses (standard deviation, percentiles, correlation coefficients, z-scores) to quantify coverage variability and detect outlier states
  • Created interactive visualizations including geographic heat maps, scatter plots with regression analysis, and waterfall charts revealing 15-40% infrastructure-to-street deployment gaps across technologies
  • Performed comparative analysis identifying territory coverage lags averaging 12-18% behind state averages, informing strategic infrastructure investment priorities
  • Engineered 5-page Power BI dashboard with synchronized slicers and drill-through capabilities, reducing analysis time by enabling dynamic filtering across 4 technology generations (3G/4G/5G Tier 1/5G Tier 2)
  • Developed Python-based web dashboard using Plotly Dash framework with real-time filtering, delivering responsive analytics accessible across devices without software installation requirements
